# 💼 실업급여 계산기 허브

대한민국 실업급여를 간편하게 계산할 수 있는 웹/CLI 계산기입니다.

## 🚀 빠른 시작

### 웹 버전 (권장)

**브라우저에서 바로 사용! 설치 불필요**

```bash
# 간단한 HTTP 서버로 실행
cd web
python3 -m http.server 8000
open http://localhost:8000

# 또는 파일로 직접 열기
open web/index.html
```

### CLI 버전 (Mojo)

**터미널에서 대화형으로 사용**

```bash
# Mojo 설치 필요
mojo src/unemployment_calculator/main.mojo
```

---

## 기능

이 계산기는 한국 고용보험 실업급여 계산 규정에 따라 다음 세 가지 유형의 근로자에 대한 실업급여를 계산합니다:

- **일반 근로자**: 정규직 및 계약직 근로자
- **일용 근로자**: 일당을 받는 단기 근로자
- **자영업자**: 고용보험에 가입한 자영업자

## 계산 항목

각 유형별로 다음 항목을 계산합니다:

- **1일 구직급여일액**: 하루에 받을 수 있는 실업급여 금액
- **예상 지급일수**: 실업급여를 받을 수 있는 총 일수 (120~270일)
- **총 예상수급액**: 전체 수급 기간 동안 받을 수 있는 총 금액

## 계산 기준

- **급여율**: 일 평균 임금의 60% (2019년 10월 1일 이후)
- **상한액**: 66,000원/일
- **하한액**: 최저임금의 80% × 8시간
- **수급 자격**: 고용보험 가입 기간 180일 이상

### 사용 예시

프로그램을 실행하면 대화형 메뉴가 나타납니다:

```

============================================================

계산하려는 유형을 선택하세요:
1. 일반 근로자
2. 일용 근로자
3. 자영업자
0. 종료

선택: 1
```

## 계산 로직 설명

### 일반 근로자

1. 최근 3개월 평균 월급 계산
2. 일 평균 임금 = 월 평균 ÷ 30일
3. 일일 수급액 = 일 평균 임금 × 60%
4. 상한액/하한액 적용
5. 나이와 가입기간에 따라 지급일수 결정

### 일용 근로자

1. 최근 3개월 총 임금 ÷ 총 근무일수 = 일 평균 임금
2. 일일 수급액 = 일 평균 임금 × 60%
3. 상한액/하한액 적용
4. 나이와 가입기간에 따라 지급일수 결정

### 자영업자

1. 기준보수 등급에 따른 월 기준보수 결정
2. 일 평균 임금 = 월 기준보수 ÷ 30일
3. 일일 수급액 = 일 평균 임금 × 60%
4. 상한액/하한액 적용
5. 가입기간에 따라 지급일수 결정

## 지급일수 결정 기준

| 가입기간 | 30세 미만 | 30-50세 미만 | 50세 이상/장애인 |
|----------|-----------|--------------|------------------|
| 1년 미만 | 120일 | 120일 | 120일 |
| 1-3년 미만 | 120일 | 150일 | 180일 |
| 3-5년 미만 | 150일 | 180일 | 210일 |
| 5-10년 미만 | 180일 | 210일 | 240일 |
| 10년 이상 | 210일 | 240일 | 270일 |

## 주의사항

- 이 계산기는 **모의 계산**을 위한 도구입니다
- 실제 수급액은 고용센터의 정확한 심사를 통해 결정됩니다
- 법률 및 정책 변경에 따라 계산 방식이 달라질 수 있습니다
- 정확한 수급액은 [고용보험 홈페이지](https://www.ei.go.kr)에서 확인하세요
